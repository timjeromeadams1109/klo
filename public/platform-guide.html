CONFERENCE
⚙

Running a Conference

Admin guide to managing live events

  • Create sessions — Name each segment (e.g., "Morning Keynote", "Panel Discussion")
  • Toggle Seminar Mode on to activate conference tools for attendees
  • Choose a release mode: Show All, Single Release, or Hide All
  • Moderate questions — mark answered, hide, archive, or restore
  • Open the Monitor View on a projector for the audience display
Pro Tip: Create all sessions ahead of time. During the event, switch between them with one tap. See the Conference V2 Guide for detailed instructions.

FAQ — Getting Started

How do I sign up?
Visit klo-app.vercel.app and click "Get Started." Sign in with your Google or Apple account — no separate password needed.
Is the platform free?
Yes! The KLO platform is free to use. Create an account to unlock all features including assessments, AI Advisor, Vault, and more.
Do I need an account to take assessments?
No — anyone can take assessments as a guest. Sign in to save results to your profile and export as PDF, Word, or PowerPoint.
What do I get by creating an account?
Saved assessment results, export reports, AI Advisor access, Vault library, Intelligence Feed, Strategy Rooms, and conference likes.
Which sign-in methods are supported?
Google and Apple OAuth. Just tap the sign-in button and authenticate with your existing account — no new password to remember.

FAQ — Staff & Admin

How do I get admin access?
The platform Owner assigns Admin roles. Contact the Owner to request access — it takes effect immediately.
What's the difference between Admin and Moderator?
Admins manage the full dashboard (users, content, analytics). Moderators focus on live conference tools (Q&A, profanity, sessions).
How do I set up a conference?
Open the Conference tab in the admin dashboard, create sessions, then toggle Seminar Mode on when you're ready to go live.
How does the profanity filter work?
A customizable word list screens Q&A and word cloud submissions. Blocked attempts are logged in an audit trail for review.
Can multiple people moderate at once?
Yes. Multiple Admins and Moderators can manage the conference simultaneously. Role changes take effect immediately.

Troubleshooting

I can't sign in
Make sure you're using the same Google or Apple account you signed up with. Clear your browser cache and try again.
My assessment won't load or save
Check your internet connection and refresh the page. If answers aren't saving, try a different browser. Progress is auto-saved as you go.
AI Advisor isn't responding
Make sure you're signed in. If the advisor is slow, wait a moment — it processes each response in real time. Try refreshing if it stalls.
Conference tools aren't showing
Conference tools only appear when Seminar Mode is active. Ask the event admin to confirm it's turned on. Try refreshing the page.
Export isn't working
Export requires a registered account. Make sure you're signed in. Disable browser pop-up blockers and try again.
