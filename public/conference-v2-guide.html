What's New in V2

A complete upgrade to the Conference Companion experience

  • Session Management — Organize content into named sessions with independent controls
  • Role-Based Access — Owner, Admin, Moderator, Registered, and Guest roles
  • Question Likes & Top 5 — Registered users can heart questions; top 5 get highlighted
  • Profanity Filter — Customizable word list with audit logging of blocked submissions
  • Soft-Delete Archive — Questions are archived, not destroyed — restorable anytime
  • Release Modes — Control exactly which questions the audience sees: all, one-at-a-time, or none

For Attendees

Joining the Conference

  • Navigate to /conference in the KLO app
  • Conference tools activate automatically when Seminar Mode is turned on by an admin
  • You'll see the active session name and available interaction panels
  • No special login required — Guest participation is supported for Q&A and Word Cloud
  • Registered users unlock additional features like question likes
Tip: Sign in to your KLO account to access the "heart" feature and see your submitted questions.

For Attendees
📊

Live Polls

  • When a poll is active, it appears on your conference screen automatically
  • Tap your choice to cast your vote — results update in real time
  • Poll results are displayed as animated bar charts visible to everyone
  • Each device gets one vote per poll to ensure fair results
  • Admins control when polls open and close
Note: Once you vote, your selection is final. Choose carefully!

For Attendees
❓

Q&A — Ask Questions

  • Type your question in the input field and tap Submit
  • Questions are anonymous by default — your name is not displayed
  • A profanity filter screens submissions — blocked words will prevent posting
  • Your question enters the moderation queue and appears once released by an admin
  • You can submit multiple questions throughout the session
Moderated Experience: Admins review questions before they appear on the main screen. This keeps the Q&A focused and productive.

For Attendees

Likes & Top 5

  • Registered users can tap the heart icon on any visible question to like it
  • Like counts are displayed in real time next to each question
  • The Top 5 most-liked questions receive a special gold badge
  • Top 5 questions are featured prominently on the Monitor View projection
  • You can unlike a question by tapping the heart again
Why it matters: Liking helps surface the most important questions for the speaker to address. The Top 5 ensures the best questions get answered.

Word Cloud

  • When a Word Cloud session is active, you'll see a text input prompt
  • Submit a single word or short phrase that captures your thought
  • The cloud updates live — popular words appear larger
  • Profanity filter applies here too — inappropriate words are blocked
  • Great for brainstorming, pulse checks, and audience engagement
Best Practice: Keep submissions to 1–3 words for the clearest visual impact on the cloud.

For Admins

Getting Started — Admin Panel

  • Access the admin panel from the Admin tab in the conference interface
  • Only users with Owner, Admin, or Moderator roles can see this tab
  • The panel provides controls for: Sessions, Questions, Polls, Word Cloud, Roles, and Settings
  • Toggle Seminar Mode on/off to control when attendees can interact
First time? Start by creating a session, then toggle Seminar Mode on. Attendees will see the conference tools appear automatically.

Session Management

  • Create a session — Give it a name (e.g., "Morning Keynote", "Panel Discussion")
  • Activate a session — Only one session is active at a time; switching is instant
  • Toggle Q&A — Enable or disable question submissions per session
  • Set Release Mode — Control how questions are shown to the audience
  • Each session maintains its own questions, polls, and word cloud data independently
Tip: Create all your sessions ahead of time. During the event, just switch between them with one tap.

For Admins
🛡

Question Moderation

  • Mark Answered — Flag questions the speaker has addressed
  • Hide — Remove a question from the audience view without deleting it
  • Archive — Soft-delete a question; it moves to the archive and can be restored
  • Release Toggle — In "Single Release" mode, release questions one at a time
  • Restore — Bring back any archived question with full like count preserved
Nothing is permanently deleted. The archive system ensures you can always recover questions. Use Archive freely to keep the Q&A clean.

Release Modes

Control exactly what the audience sees

🔍

Show All

Every approved question is visible to the audience immediately. Best for open, informal sessions.

☝

Single Release

Release questions one at a time. The speaker addresses each before the next appears. Best for structured Q&A.

🚫

Hide All

No questions are shown to the audience. Useful during presentations or breaks. Questions still collect in the queue.

Profanity Filter

  • View current word list — See all terms currently being filtered
  • Add terms — Type a word or phrase to add it to the block list
  • Remove terms — Delete words that were added by mistake
  • Audit log — Every blocked submission is logged with timestamp and content
  • Filter applies to both Q&A submissions and Word Cloud entries
Audit Log: Check the log periodically during events to see if attendees are attempting inappropriate submissions. The log helps you decide if additional terms should be added.

For Admins
👥

Role Management

  • Owner — Full platform access: all admin capabilities plus system-level controls
  • Admin — Full conference access: sessions, moderation, roles, settings, profanity filter
  • Moderator — Can moderate questions, manage sessions, view audit logs, monitor view
  • Registered — Can vote, ask questions, submit words, like questions, view monitor
  • Guest — Can vote in polls, ask questions, and submit word cloud entries
  • Assign roles by user ID from the admin panel
  • Role changes take effect immediately — no refresh required
  • Multiple admins and moderators can manage simultaneously
  • Roles persist across sessions within the same event

Monitor View

Full-screen projection for the big screen

  • Displays the Top 5 questions ranked by like count
  • Auto-cycling — Questions rotate automatically for continuous display
  • Shows the active session title prominently at the top
  • Designed for projectors and large screens — high contrast, large text
  • Updates in real time as likes change and new questions are released
Setup: Open the Monitor View URL on the projection computer and go full-screen (F11). It runs independently — no interaction needed once started.
