Your KLO platform is built and live with 33 pages, AI-powered advisory, assessments, content vault, marketplace, and Stripe billing. To take the site fully live in production, we need the following items from you. Please check off each item and provide the requested assets or information.

1. Accounts & Credentials

API keys and account credentials needed for production services.

  • Stripe Account & Price IDs Required
    Create products in your Stripe Dashboard for Pro ($29/mo) and Executive ($99/mo) plans. Provide the Price IDs (e.g., price_1Abc123...) for each. Also provide your STRIPE_SECRET_KEY and STRIPE_WEBHOOK_SECRET.
    Sign up at: dashboard.stripe.com
  • Supabase Project Credentials Required
    Create a Supabase project and provide SUPABASE_URL and SUPABASE_ANON_KEY. This stores user profiles, assessment results, and subscription data.
    Sign up at: supabase.com
  • Google OAuth Client ID & Secret Recommended
    For "Sign in with Google" functionality. Create OAuth credentials in Google Cloud Console. Provide GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET.
  • Resend API Key Recommended
    For transactional emails (welcome, assessment reports, booking confirmations). Provide RESEND_API_KEY. Also verify your domain (keithlodom.io) in Resend.
    Sign up at: resend.com
  • NextAuth Secret Required
    A random secret string for encrypting auth sessions. We can generate this for you. Provide NEXTAUTH_SECRET (run: openssl rand -base64 32).
